# sales-management
## **_Software for sales management and inventory control_**

## Features

- Customer base
- Register products
- Register sale
- Daily production
- Export and import data backup

## Run

With node 16 or higher installed:

```sh
npm install
npm run dev
```

This project uses nextron as a base: <https://github.com/saltyshiomix/nextron>
